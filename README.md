<h3>1ï¸âƒ£ Scatter Plot</h3>
<p>
The scatter plot is used to identify the relationship between two numerical variables.
Each point represents an observation, helping to detect patterns, correlations,
and outliers in the data.
</p>
<ul>
  <li>Shows relationship between variables</li>
  <li>Helps identify trends and outliers</li>
  <li>Useful for correlation analysis</li>
</ul>

<h3>2ï¸âƒ£ Line Chart</h3>
<p>
The line chart is used to visualize trends over time.
It helps in understanding how values change continuously across a timeline.
</p>
<ul>
  <li>Best for time-series analysis</li>
  <li>Shows upward or downward trends</li>
  <li>Easy comparison across periods</li>
</ul>

<h3>3ï¸âƒ£ Area Chart</h3>
<p>
The area chart is similar to a line chart but with the area below the line filled.
It emphasizes the magnitude of change over time.
</p>
<ul>
  <li>Highlights volume and growth</li>
  <li>Effective for cumulative data</li>
  <li>Improves visual impact of trends</li>
</ul>
